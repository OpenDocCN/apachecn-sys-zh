## 下载

### Docker

```
docker pull apachecn0/apachecn-cs-zh
docker run -tid -p <port>:80 apachecn0/apachecn-cs-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install apachecn-cs-zh
apachecn-cs-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-cs-zh
apachecn-cs-zh <port>
# 访问 http://localhost:{port} 查看文档
```
